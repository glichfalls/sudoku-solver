# Sudoku (solver) App

## Features
- import sudoku from a text file
- import sudokus in JSON format generated by http://www.cs.utep.edu/cheon/ws/sudoku/new/  
- auto solve sudokus
- solve sudokus by yourself
- auto generate random sudokus
